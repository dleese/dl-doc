= Home
The Home screen is the main entry point of Logipad. From here you can sign in, synchronize operational data, review status information, and open commonly used items.

[IMPORTANT]
====
Operational notice::
Logipad supports the operator’s EFB programme. Always follow your operator’s procedures for login, data currency checks, and synchronization prior to flight. If any information in the app appears outdated or inconsistent, re-synchronize and follow operator escalation procedures.
====

The Home screen is typically divided into these areas:

* Login and *Synchronize*
* User information (account and logout)
* Status and notifications
* Favorites and history
* News (if enabled by the operator)

== Login/Synchronize
After login, the navigation menu may show badges. A red badge indicates new or updated content.

[.center-caption]
.Home screen — navigation menu with badges
image::ios_home_menu.png[Logipad Home screen navigation menu,75%,float="center",align="center"]

Tap *Synchronize* to download the latest operational data to the device. Synchronization ensures the content on the EFB matches the current server state (subject to connectivity and operator configuration).

[WARNING]
====
Data currency::
Do not rely on cached data alone. Confirm a successful synchronization in accordance with your operator’s procedures before using Logipad for operational flight purposes.
====

== User Information
The User Information area shows the signed-in account. To log out, tap the user information area, then confirm by tapping btn:[Logout].

[.center-caption]
.User Information area — logout confirmation
image::ios_logout.png[Logipad logout confirmation,50%,float="center",align="center"]

== Content and Information view
The Status and Notifications area shows operational status information and messages relevant to the current user. The exact content depends on operator configuration and available connectivity.

== Favorite and History view
The Favorites and History area provides quick access to:

* *Favorites* — items you marked for fast access.
* *History* — recently opened items (for example, documents opened in the Documentation module).

[.center-caption]
.Favorites and History view
image::ios_Favorites_History.png[Logipad Favorites and History view,50%,float="center",align="center"]

== News Section View
If enabled, the News section displays announcements and updates published by the operator (for example operational messages or brief notices). Treat operator-published information as guidance only; cross-check against the operator’s controlled documentation when required.
