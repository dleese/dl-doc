= Module Library

The Library module is a core component of Logipad for document management and role-based provisioning. It supports the organization, classification, and controlled distribution of documents using configurable attributes and options.

== Capabilities

* **Role-based provisioning** — Assign documents to content roles for targeted distribution to crews or roles.
* **Document flags** — Mark documents with one or more of:
** Confirmable
** Exportable
** Indexable
* **Priority** — Assign priority levels to documents.
* **Open behavior** — Choose whether a document opens in the Logipad application or in an external application.
* **Validity period** — Publish documents in a time-controlled manner using "Valid from" and "Valid until".
* **Comments** — Add internal descriptions or notes to documents.
* **Inventory** — Enable inventory handling when a document is opened.

=== Supported content types

The following content types can be published:

* video
* txt
* epub
* zip
* pdf
* htmlzip
* html
* doc
* docx
* xls
* xlsx
* ppt
* pptx

== Document organization

Documents are organized in a folder structure. Folders can be associated with a document template to apply standard properties to documents placed in that folder. This keeps metadata and settings consistent and simplifies document ingestion.
